Wrist Reader

A New Way for the Blind and Deaf to Interact With People and Information!

Wrist Reader is a device that takes characters of a language and outputs them to vibrating motors onto the wrist of the user. The motors are placed into a geometric shape, with each character(including numbers and symbols) having its own unique vibration pattern.

Made With: arduino processing

Github Link: github

Gift Buddy

Ever Been Shopping for Someone and You Forgot What Size They Are? Gift Buddy Is Your New Best Friend...err I Mean Buddy.

Gift Buddy is a simple but complete app that allows you to save clothing information on all the people that you buy clothes for. Design is minimal and familiar to all iOS users. The aim of the app was to be reliable basic and unassuming, the user doesn't need to be distracted by unnecissary design polish. But maybe I'm just saying that because this was one of the first apps I've made on iOS and couldn't add design polish if I tried? Either way, I think it's pretty neat!

Made With: xcode objective-c

Github Link: github

Clock-Steezy

The Watch-face With Steez!

Clock-Steezy is a watch-face prototype for the Pebble smart watch. It features a unique representation of time and presents it to you with a pleasing design.

Made With: processing

Github Link: github

SineTron

Come On, Do the Wave!

SineTron is a game that is based on a sine wave. The screen is filled with balls that are attacted to the motion of the sine wave. Players avoid touching them while the balls are white and collect points touching them while they are colored. It's simple, addictive fun!

Made With: processing spritebuilder xcode swift

Github Link: github github
